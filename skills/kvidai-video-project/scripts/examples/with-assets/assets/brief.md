# Video Brief

**Goal**: Create a 30-second vertical video for YouTube Shorts (9:16, 1080×1920).
**Tone**: Professional yet engaging.

## Script

1. (0-5s) Open with the reference image as the hero visual — animate with a slow zoom-in.
2. (5-25s) Cut to B-roll footage with voiceover narration describing the main message.
3. (25-30s) Close with a bold text CTA overlay and brand-color background.

## Narration style

Confident, clear. Prefer Korean narration. End with a short English tagline.
